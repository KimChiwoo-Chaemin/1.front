HTML/CSS 학습 중, 게임 UI에 관심 있음

사용자가 보기 편한 화면을 만드는 것에 관심이 많다 시맨틱 태그를 사용해서 구조를 명확하게 작성하려고 노력하고 있으며 간단한 레이아웃, 텍스트 스타일링, 배경, 그림자 효과 등을 구현할 수 있다

기술 스택/ 사용 가능 기술

HTML

  • 시맨틱 태그
  • 이미지, 링크 단락

CSS

  • 테두리, 배경 할 수 있음
  • 그림자, 배치 할 수 있음

프로젝트

시맨틱 태그와 CSS 기본 스타일만으로 개인 포트폴리오 페이지를 제작했다. 박스 모델, 여백 ,테두리 , 그림자 등을 이용하여 카드 형태의 UI를 구성했고 flex를 이용해 프로필 영역의 사진과 텍스트를 가로로 정렬했다.
